# BC-realne-roboticke-dvojča-FANUC

Tento repozitár obsahuje projekt **HandlingPRO4**  v prostredí **FANUC ROBOGUIDE**. Ide o simulačný program, ktorý vychádza z back up-u programu z reálneho priemyselného robota.
Slúži na virtuálne testovanie, ladenie trajektórií a overovanie správania robota mimo reálnej výroby.

## repozitár obsahuje
- hlavný projekt - HandlingPRO4.frw
- virtuálneho robot - Robot_1
- robotické programy a registre - TP/VR napr. (con_pant_v5.tp) (`numreg.vr` a `posreg.vr`)
- layoutu pracoviska - layout
- modelov uchopovača - 3jaw_Gripper_jaw.csb
- doplnkové dáta


## Cieľ
- práca s robotickým programom mimo reálnej výroby
- simulácia pohybov a trajektórií robota
- testovanie správania pracovného cyklu
- kontrolu pozícií, registrov a pracovných bodov
- ladenie programu vo virtuálnom prostredí
- overovanie zmien pred ich použitím na reálnom zariadení
